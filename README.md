# Cycle JSONP Driver

A [Cycle.js](http://cycle.js.org) [Driver](http://cycle.js.org/drivers.html) for making HTTP requests through the JSONP hack, based on the [jsonp](https://github.com/webmodules/jsonp) package.

```
npm install @cycle/jsonp
```
